## Introduction

SystemVerilog has become the standard HDL for modern FPGA development, offering both simulation and synthesis capabilities. This post covers proven design patterns that improve code quality, readability, and synthesizability.

## Parameterized Modules

Parameterization enables reusable components across different designs:

```verilog
module fifo #(
    parameter WIDTH = 8,
    parameter DEPTH = 16
) (
    input clk, rst,
    input [WIDTH-1:0] data_in,
    input write_en, read_en,
    output [WIDTH-1:0] data_out,
    output full, empty
);
    // Implementation
endmodule
```

## Finite State Machines

Using `enum` types for FSM states improves clarity:

```verilog
typedef enum logic [2:0] {
    IDLE = 3'b000,
    ACTIVE = 3'b001,
    DONE = 3'b010
} state_t;

always_ff @(posedge clk or negedge rst) begin
    if (!rst)
        current_state <= IDLE;
    else
        current_state <= next_state;
end
```
