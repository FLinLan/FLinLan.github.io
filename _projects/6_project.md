## Project Overview

Implemented a programmable feedback and feedforward control loop that accepts a 0.3V to +0.3V signal and amplifies it to 15V while performing stability control within the STM32G4 microcontroller. This board demonstrates advanced signal conditioning and real-time control implementation.

## Technical Specifications

- **Input Signal Range:** 0.3V to +0.3V
- **Output Voltage:** 15V
- **Microcontroller:** STM32G4 MCU
- **Control Function:** Programmable feedback/feedforward loop
- **Stability Control:** Real-time feedback monitoring and adjustment

## Key Features

- **Signal Amplification:** 50× gain amplification from 0.3V input to 15V output
- **Control Loop:** Programmable feedback and feedforward configurations
- **Stability Management:** Real-time control loop stability monitoring within MCU
- **Programmability:** Flexible control parameters via STM32G4 firmware

## Accomplishments

- Implemented programmable feedback and feedforward control loop for signal amplification
- Designed signal conditioning circuit accepting 0.3V to +0.3V input with 50× amplification to 15V output
- Integrated STM32G4 MCU for real-time stability control and feedback monitoring
- Implemented firmware control logic for adjustable feedback and feedforward gains
- Verified circuit stability and performance through practical testing and measurement
- Successfully demonstrated dynamic control loop operation with predictable and stable output
