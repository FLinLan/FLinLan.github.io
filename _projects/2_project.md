## Overview

Designed a 2×2 microstrip patch antenna array operating at 4.5 GHz using FR-4 substrate with comprehensive RF analysis and optimization in Ansys HFSS. The antenna features a corporate feeding network with quarter-wave impedance transformers for optimal power transfer.

## Key Features

- **Operating Frequency:** 4.5 GHz
- **Substrate:** FR-4
- **Configuration:** 2×2 patch antenna array
- **Impedance Matching:** Corporate feeding network with quarter-wave transformers
- **Analysis Tool:** Ansys HFSS

## Design Parameters

- **Input Impedance:** 54.8 ohm (optimized for 50 ohm matching)
- **Peak Gain:** 8.52 dBi
- **Beamwidth:** 43° × 44°
- **Radiation Efficiency:** 0.69 (69%)

## Accomplishments

- Designed antenna array in Ansys HFSS using FR-4 substrate with transmission-line model derivation for patch dimensions
- Implemented corporate feeding network with quarter-wave impedance transformers for 50 ohm impedance matching
- Analyzed S11 parameters, VSWR, and input impedance to optimize total RF power transfer
- Verified far-field behavior in Ansys, achieving excellent performance metrics: 8.52 dBi peak gain, 43 × 44 degree beamwidth, and 0.69 radiation efficiency
