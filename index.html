FPGA Taiko No Tatsujin

Short summary about my second project

  • Designed an FPGA-based rhythm game on DE1-SoC using Verilog, integrating VGA display and 7-segment score output for real-time gameplay.
  • Implemented note animation with 16-bit hex file reads at 3 Hz, achieving 100% rhythm accuracy for 130 frames.
  • Optimized collision detection logic, reducing input-to-score latency to 1 ms via one-hot encoded 16-bit frame checks.
  • Developed FSM for game state transitions, verified with ModelSim testbenches, achieving 95% code coverage.
  • Utilized Intel Quartus for synthesis, leveraging 50% of Cyclone V memory (1.66 Mb) for smooth VGA rendering.
